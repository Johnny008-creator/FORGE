# Forge v0.5.0 — Full Agent Transition

## 🎯 Release Highlights

### 1. 📂 Windows Folder Management (`mkdir`)
Forge now includes a native `mkdir` tool, allowing it to safely create complex directory structures using `os.makedirs(exist_ok=True)`. This resolves previous issues where tiny models struggled to organize projects into folders.

### 2. 🧠 Smart Context Compacting (`/compact`)
Small models (0.5B - 3B) often suffer from limited context windows. The new `/compact` command:
- **Summarizes** the previous conversation history into a single paragraph.
- **Frees up** context space while preserving the core task status.
- **Improves** long-term task completion on lightweight hardware.

### 3. 📊 Dynamic Context Detection
Forge no longer relies on hardcoded message limits. It now:
- **Queries Ollama API** for the exact `context_length` of each model.
- **Estimates Token Usage** to perform smarter message trimming.
- **Adjusts Dynamically** when you switch models on the fly.

### 4. 🌐 Global English Transition
To ensure maximum compatibility and model reliability:
- **System Prompts** have been optimized and standardized in English.
- **Full Codebase** (comments, docstrings, logs) has been translated.
- **Instruction Accuracy** has been improved by using clearer English terminology.
