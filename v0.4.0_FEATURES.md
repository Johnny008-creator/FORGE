# Forge v0.4.0 — Auto Model Detection & Animations

## 🎯 What Changed

### 1. Real Model Detection from Ollama API

**Before (v0.3):** Regex parser on model name
```
qwen2.5:0.5b  →  [regex]  →  0.5B  →  assume tiny profile
```

**After (v0.4):** Direct from Ollama API
```
Ollama /api/tags
{
  "parameter_size": "494.03M",    ← real data
  "quantization_level": "Q4_K_M",
  "family": "qwen2",
  "context_length": 32768
}
                ↓
Model dict {
  params_b: 0.49,
  quant: "Q4_K_M",
  family: "qwen2",
  size_mb: 379,
  context_length: 32768
}
                ↓
Model picker shows:
  qwen2.5:0.5b    0.49B    Q4_K_M    379MB    qwen2
```

**Benefits:**
- No guessing — actual parameters from Ollama
- Supports ANY quantization level (Q2_K, Q4_K_M, Q8_0, etc.)
- Handles models not matching "XbB" naming pattern
- Shows quant level (affects quality/speed trade-off)

### 2. Claude Code-Like Animations

**Spinner (loading/connecting):**
```
Before: Braille spinner (crashes on Windows)
After:  Rich Spinner("dots") with auto-fallback to ASCII

  ⠹
  
  — connecting to ollama...    ← Windows fallback
```

### 3. Windows Unicode Safety

**New function: `supports_unicode()`**
- Detects terminal capability
- Returns bool for animation decisions
- Falls back gracefully

**Fallback mappings:**
```
⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏  →  |/-\|/-\
⟳                    →  ~
✓                    →  [OK]
✗                    →  [ERR]
⣾⣽⣻⣷⣯⣟⡿⢿      →  -\|/-\|-
```

# Forge v0.5.0 — Windows mkdir & Context Management (dev)

## 🎯 What's New

### 1. New `mkdir` Tool
- **Safely create directories**: Uses `os.makedirs(exist_ok=True)`.
- **Windows Support**: Robust handling of backslashes and path structures.
- **Few-shot examples**: Added to system prompt for better reliability on tiny models.

### 2. Conversation Compacting (`/compact`)
- **Problem**: Tiny models (0.5B) have very small context windows (4-6 messages).
- **Solution**: The `/compact` command uses the LLM to summarize previous messages into a single paragraph.
- **Result**: Frees up context space while preserving the "essence" of the conversation.

### 3. Dynamic Context Length from API
- **Better Accuracy**: Uses `context_length` directly from Ollama's `/api/tags` if available.
- **Heuristic Fallback**: Smart defaults based on model parameter size if API doesn't provide it.

### 4. Language Support
- **English-First**: Documentation and system prompts optimized for English for maximum compatibility across models.
